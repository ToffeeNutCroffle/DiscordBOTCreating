## v0.6 — 2026-04-12

### 추가
- 일자 갱신 시간을 오전 4시로 변경 — 자정~오전 4시 사이의 활동은 전날 개발일로 집계

### 개선
- 개발 통계, 개발 달력, 개발 랭킹에서 진행 중 세션 실시간 반영 복구
- 개발 랭킹 조회 쿼리 최적화 — 다수가 동시에 개발 중일 때 응답 속도 개선

## v0.5 — 2026-04-11

### 개선
- 개발 달력 개발일 표시를 초록색 숫자에서 ✓ 문자로 변경 (모바일 정렬 깨짐 방지)
- 개발 통계, 개발 달력, 개발 랭킹에서 진행 중 세션 실시간 반영 제거 — 퇴장 후 기록된 데이터만 표시

### 버그 수정
- 개발 달력 ✓ 표시에 초록색 ANSI 색상이 적용되지 않던 문제 수정

---

## v0.4 — 2026-04-09

### 개선
- 개발 달력 요일 순서를 일·월·화·수·목·금·토 순으로 변경
- 개발 달력에 ANSI 색상 적용 — 일요일 빨간색, 토요일 파란색, 개발한 날 초록색
- 개발 랭킹에서 표시 순서를 `시간 / 일` 순으로 변경 (기존: `일 / 시간`)

### 버그 수정
- 개발 달력에서 개발한 날 표시 시 UI가 밀리는 문제 수정

---

## v0.3 — 2026-04-08

### 개선
- 배포 환경을 로컬 하드웨어에서 fly.io 클라우드로 이전

### 버그 수정
- 자정을 넘기는 세션의 날짜 집계 버그 수정
- 개발 랭킹 개발일수를 90분 이상인 날만 집계하도록 수정
- 개발 랭킹에서 진행 중인 세션의 개발일수도 실시간 반영

## v0.2 — 초기 기능 확장

### 추가
- 개발 랭킹에서 진행 중인 세션 시간 실시간 반영
- `/사용법` 커맨드 추가
- 슬래시 커맨드를 개발실 카테고리에서만 사용 가능하도록 제한
- 오늘 개발시간 항상 표시

### 개선
- 봇 응답을 모두에게 보이도록 변경 (기존: 나만 보기)
- 개발 랭킹 시간 집계를 sessions 테이블 기반으로 통일

### 버그 수정
- 이번 달 개발시간·개발일 실시간 반영 버그 수정
- 통계 계산 버그 수정
