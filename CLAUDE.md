# 디스코드 개발 추적 봇

## 프로젝트 개요
서버 내 '개발실' 분류의 음성채팅방 잔류 시간을 추적하여 개발 활동을 기록하고 통계를 제공하는 디스코드 봇

## 핵심 기능
- **출석 기록**: '개발실' 분류 음성채팅방에 1시간 이상 잔류 시 해당 날짜를 개발일로 기록
- **연속 개발일**: 현재 며칠 연속으로 개발 중인지 표시
- **월별 개발일수**: 이번 달에 개발한 총 일수 표시
- **월별 개발시간**: 이번 달에 개발한 총 시간(분 단위까지) 표시

## 기술 스택
- **언어**: Python
- **라이브러리**: discord.py
- **데이터베이스**: SQLite (로컬) 또는 추후 결정
- **배포 환경**: 별도 하드웨어 (로컬 머신 아님) — 환경변수 기반 설정 필요

## 프로젝트 구조 (예정)
```
DiscordBOTCreating/
├── bot.py              # 봇 엔트리포인트
├── cogs/               # 기능별 모듈
│   ├── tracker.py      # 음성채널 추적 로직
│   └── stats.py        # 통계 조회 커맨드
├── database/
│   └── db.py           # DB 연결 및 쿼리
├── .env                # 토큰 등 민감 정보 (git 제외)
├── .env.example        # 환경변수 템플릿
└── requirements.txt
```

## 주요 규칙
- `.env` 파일은 절대 커밋하지 않는다 (토큰, DB 경로 등 민감 정보 포함)
- 음성채널 이벤트는 `on_voice_state_update`로 처리
- 1시간 미만 잔류는 기록하지 않음
- 날짜 기준은 서버 로컬 시간이 아닌 **KST (UTC+9)** 기준으로 통일

## 환경변수 목록 (.env.example 기준)
```
DISCORD_TOKEN=your_bot_token_here
DEV_CATEGORY_NAME=개발실
MIN_SESSION_MINUTES=60
```
